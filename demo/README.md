# cross-tab Demo

Interactive demo showcasing all features of `cross-tab`.

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

The demo will open at `http://localhost:3000`

## Features Demonstrated

1. **Theme Synchronization** - Change theme in one tab, see it update in all tabs
2. **Counter** - Simple counter that syncs across tabs
3. **Shopping Cart** - Complex state with add/remove operations
4. **Authentication** - Login/logout sync across tabs
5. **Persistence** - Compare persisted vs non-persisted state
6. **Custom Serialization** - Handle complex data types (Date, Map, Set)

## Try It Out

1. Open the demo in multiple browser tabs
2. Interact with the examples
3. Watch state synchronize in real-time across all tabs!
